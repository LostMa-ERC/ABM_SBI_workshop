# \[WORKSHOP\] Agent-based Models and Simulation based inference for manuscript transmission

This repository contains the necessary code for the workshop dedicated to agent-based modeling and simulation-based parameter inference.

- The first part of the workshop is dedicated to the [simMAtree](https://github.com/LostMa-ERC/simMAtree) python package, that can be installed by running

```shell
pip install git+https://github.com/LostMa-ERC/simMAtree.git
```

- The second part will be dedicated to a direct use of the `sbi` python package for parameter inference from topological properties of stemmata, using (`notebook.ipynb`, [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/LostMa-ERC/ABM_SBI_workshop/blob/main/notebook.ipynb))
